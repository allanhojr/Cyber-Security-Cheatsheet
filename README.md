# 🛡️ Ethical Hacking Cheat Sheet

A dark terminal-themed cheat sheet covering the full penetration testing lifecycle — 
built with vanilla HTML, CSS, and JavaScript.

## What's covered
- Passive & Active Reconnaissance (OSINT, DNS, Shodan, Google Dorks)
- Scanning & Enumeration (Nmap, SMB, LDAP, FTP, SNMP)
- Web Application Testing (SQLi, XSS, LFI, SSRF, fuzzing)
- Password Attacks (Hydra, Hashcat, John the Ripper)
- Exploitation (Metasploit, reverse shells, msfvenom)
- Post-Exploitation & Privilege Escalation (Linux & Windows)
- Network Pivoting, Wireless Attacks, Defensive Concepts

## Features
- One-click copy button on every command
- Mobile-friendly (icon-only copy button on small screens)
- Color-coded by attack category
- No dependencies — pure HTML/CSS/JS

## Usage
Clone the repo and open `index.html` in any browser. No server needed.
